# UsefulWeb
- 그동한 일하면서 배운 기술을 사용하여 넣은 잡다한 게시판
- DB연동 mybatis사용
- SSH 접근 설정(정식으로 발급받은 키는 아님.)
- Application.properties 암호화, 외 SHA256을 이용하여 민감한 유저정보 암호화 (암호화키는 숨김)
- 유저정보 세션 Vo에 담아 쿠키로 보냈지만 토큰으로 관리할지 고민중.
